# Delimited Text Extractor

A .NET 8 minimal API that extracts text wrapped in parentheses, brackets, braces, or single/double quotes. Upload `.txt`, `.docx`, `.pdf`, or `.csv` files and download grouped results directly in the browser.

## Run locally

```bash
cd apps/delimited-text-extractor
dotnet run
```

Then open `http://localhost:5000` (or the port shown in the console) to use the UI.
